# open-notebook-desktop

Dieser Fork verweist auf das **Desktop-Launcher-Projekt** für [Open Notebook](https://github.com/lfnovo/open-notebook).

## Desktop-Launcher (Tauri, Linux)

Der native Linux-Desktop-Launcher — verwaltet den Docker-Stack und öffnet Open Notebook in einem eigenen Fenster — liegt in einem **eigenen Repository**:

### 👉 [3ddruck12/open-notebook-launcher](https://github.com/3ddruck12/open-notebook-launcher)

- **Quellcode & Issues:** [github.com/3ddruck12/open-notebook-launcher](https://github.com/3ddruck12/open-notebook-launcher)
- **Releases (AppImage, .deb):** [Releases](https://github.com/3ddruck12/open-notebook-launcher/releases)

## Open Notebook (Web-App, Upstream)

Die eigentliche Open-Notebook-Anwendung (Backend, Frontend, Docker-Images) stammt von:

**[lfnovo/open-notebook](https://github.com/lfnovo/open-notebook)**

Der Desktop-Launcher nutzt die offiziellen Docker-Images (`lfnovo/open_notebook`) und ist ein separates Projekt.
